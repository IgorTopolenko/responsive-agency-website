# Responsive Agency Landing Page

A modern responsive landing page built with HTML and CSS for a fictional digital agency.

## Features

- Responsive layout for desktop and mobile devices
- Smooth scrolling navigation
- Interactive hover effects
- CSS animations
- Clean and structured user interface

## Technologies

- HTML5
- CSS3
- Flexbox
- Responsive Design

## Project Goal

The purpose of this project was to practice modern frontend development techniques and create a visually appealing responsive website.
